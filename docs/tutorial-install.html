Install Hikaru

Dependences

Node.js

Hikaru is written in JavaScript, so you should install Node.js.

Git (optional)

If you want to install themes with git, you need to install it.

Install

If you have installed dependences, you can install Hikaru with NPM:

# npm i -g hikarujs

Hikaru is a command line program and it provides a binary called hikaru, so you need to install it globally as root.
